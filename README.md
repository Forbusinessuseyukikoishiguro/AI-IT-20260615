# 🐰 うさうさ宇宙船ブリッジ・スイート PRO

**AI研修 導入ロードマップ・ナビゲーター ＋ ケイパビリティマップ** ／ `v5.0.0 PRO`

> **結論**：AI研修の導入を「6フェーズ」で入力するだけで、各フェーズの**実務ドキュメント**と**Claude投入プロンプト**を自動生成し、**MD/TXT/PDF/CSV**で書き出せる単一HTMLツールです。外部依存ゼロ・オフライン動作・外部送信なし。ダブルクリックで起動します。

## 📌 これは何？

- AI研修の標準ロールアウト（現状分析 → ケイパビリティマップ → 教材設計 → パイロット → 標準化 → 継続改善）を、入力フォームに沿って埋めると成果物が出てくる**設計支援ツール**。
- PH-02には、Cursor／Claude Code／マルチエージェント／RAG の**標準スキルセット（初心者向け・実務向け）**を内蔵。3層（知識・実践・設計思想）で読み・自己チェックし、設計タブへワンタップ投入できます。
- 単一HTMLファイル（vanilla JS・依存ライブラリなし）。社内配布・受講生配布にそのまま使えます。

## 🛠 PH-02 ケイパビリティマップ

2つのタブで構成されます。

- **✍️ 設計する**：プリセット（初心者向け／実務向け）から3層スキルと標準ルーブリック（L1〜L4）を**投入**し、案件向けに編集。投入は**追記方式**（既存行は消えず、同名重複なし）。
- **📖 学ぶ（マップ）**：🌱初心者／🛠実務タブで、各ツールを「**定義 → たとえ🍜 → 詳細 → 一言**」と3層で読み、**実践チェック**で自己評価。`⊕この内容を「設計する」に投入`で設計タブへ反映。

> ⚠️ ラーメン店🍜のたとえは入口理解用で **※不正確** です。各ツールの仕様は更新されるため、必ず公式ドキュメントで最新を確認してください。

## 💾 保存とモード

このファイルは**環境を自動判定**して動作します（現在のモードは画面上部ステータスと🩺ログ先頭に表示）。

- **通常モード**（ブラウザで開く／ダウンロード運用）：localStorageで**自動保存**。一次保存も端末に残り、次回復元可。**研修運用の本番向け**。
- **セッションモード**（保存制限環境）：自動保存なし（リロードで消える）。**JSON保存**や**一次保存（セッション内）**で保全。

> 確実な保全・別端末への移行は **JSON保存** を併用してください。

---

## 🩺 エラーログ

右上 **🩺 ログ**。実行時エラー（`window.onerror`等）と主要操作を記録します。エラー発生時はボタンに件数バッジが付きます。不具合時は**全コピー**または**.txt保存**で共有してください（版・モード・UA等の環境情報を自動付与）。

---

## ❓ FAQ

- **入力が消えた** → セッションモードはリロードで消えます。JSON保存か一次保存を。通常モードは同じブラウザで自動復元。
- **PDFが真っ白／全体が出る** → 印刷プレビューで「背景のグラフィック」をOFFにして保存。難しければMDで出力後、各自でPDF化を。
- **CSVが文字化け** → Excel前提のBOM付きです。スプレッドシートではUTF-8指定で開けます。
- **ボタンが反応しない** → 🩺ログを確認し、内容を共有してください。
